## ## 时间操作
### 更新操作
~~~
Timer::instance("2023-02-10")
            ->inc(1,'m') 
            ->dec(2,'d')
            ->end();
//输出 2023-03-08     
~~~

### 比较
~~~
 ## 算两个日期相差 年 月 日
 $res = Timer::instance()->diffDate("2023-01-02");
 //返回array(6) { ["y"]=> int(0) ["m"]=> int(1) ["d"]=> int(1) ["h"]=> int(0) ["i"]=> int(0) ["s"]=> int(0) }
~~~
